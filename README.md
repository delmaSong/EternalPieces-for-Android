Eternal Pieces for Android
==========================

What is
-------
원하는 스타일의 도안을 쉽게 찾아볼 수 있고 예약할 수 있는 타투 플랫폼 어플


Functions
---------
회원가입 
- 타티스트/타투어로 분기해 가입
- 타티스트의 경우 시술장소, 프로필사진, 간단한 소개, 시술가능한 요일 및 시간정보 입력받음
